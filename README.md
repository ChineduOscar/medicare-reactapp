# MediCare 🏥

A modern and responsive healthcare website built with React, TypeScript, and Vite.  
This project provides users with a clean interface to explore healthcare services, book appointments, and connect with medical professionals easily.

---

## 🚀 Live Demo

https://mymedicares.vercel.app/

---

## ✨ Features

- Responsive design for all screen sizes
- Modern healthcare UI
- Fast performance with Vite
- Built with React + TypeScript
- Smooth navigation and animations
- Fully functional newsletter subscription
- Fully functional contact sections
- Reusable components
- Clean folder structure
- API integration support

---

## 🛠️ Tech Stack

- React.js
- TypeScript
- Vite
- Tailwind CSS
- Node.js
- Express.js
- MongoDB

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/ChineduOscar/medicare-reactapp.git
```

Move into the project folder:

```bash
cd medicare
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
